# Project Planner Gantt

Ce projet est une petite application React (créée avec Vite) permettant de :
- créer des projets
- définir leurs dates
- visualiser le planning sous forme de diagramme de Gantt

## 🚀 Déploiement

Le projet est déployé automatiquement grâce à Vercel.
Je n'ai rien à faire : chaque fois que j'envoie des fichiers sur GitHub, Vercel met à jour le site.
